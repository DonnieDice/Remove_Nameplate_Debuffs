## 🚀 Quick Start

### 1. **Download**
Get RND from any of these platforms:
- [🔥 CurseForge](https://www.curseforge.com/wow/addons/remove-nameplate-debuffs) *(Recommended)*
- [📦 GitHub](https://github.com/donniedice/Remove_Nameplate_Debuffs)
- [🎯 Wago.io](https://addons.wago.io/addons/remove-nameplate-debuffs)
- [⚡ WoWInterface](https://www.wowinterface.com/downloads/info26320-RemoveNameplateDebuffs.html)

### 2. **Install**
Extract to your WoW AddOns directory:
```
📁 World of Warcraft/
  📁 _retail_/Interface/AddOns/          (The War Within)
  📁 _classic_era_/Interface/AddOns/     (Classic WoW)
  📁 _classic_/Interface/AddOns/         (Classic Cataclysm)
```

### 3. **Activate**
1. Restart World of Warcraft
2. Enable **Remove Nameplate Debuffs** in your AddOns menu
3. Type `/rnd help` to see all commands!

## 🛠️ Advanced Configuration

### Settings Persistence
All your settings are automatically saved and restored between gaming sessions using WoW's SavedVariables system.

### Default Settings
```lua
enabled = true           -- Addon is active
showWelcome = true       -- Show welcome message on login
firstRun = true          -- Show community message on first run
```

### Troubleshooting
- **Debuffs still showing?** Try `/rnd test` to refresh nameplates
- **Wrong language?** The addon auto-detects your WoW client language
- **Settings not saving?** Ensure you have write permissions in your WoW directory
