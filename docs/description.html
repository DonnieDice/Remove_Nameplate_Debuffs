# 🎯 RGX | Remove Nameplate Debuffs ![RND Logo](https://raw.githubusercontent.com/donniedice/remove_nameplate_debuffs/main/images/rnd_logo.png) ## ![](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) RGX Mods - RealmGX Community Project *** ## 🎯 Overview **RGX | Remove Nameplate Debuffs** removes all debuff icon clutter from enemy nameplates in World of Warcraft, giving you a clean, focused view of your targets. Zero configuration required — it works immediately after installation. ![RealmGX Kiwi](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) **The Kiwi Says:** "Clean nameplates, clear focus! Bwwiiiee." *** ## ✨ Features 🎯 Clean Nameplates - Removes all debuff icon clutter from enemy nameplates instantly! ⚡ Lightweight - Minimal performance impact — maximum visual clarity. 🚀 Zero Configuration - Works immediately after installation. No setup required! 🔧 Whitelist System - (Coming Soon) Choose which debuffs to keep visible. 🛡️ Plater Compatible - (In Development) Full integration with Plater addon. 🎮 Multi-Version - Works across all WoW versions — Retail, Mists, Cataclysm, and Classic Era! 🌍 Multi-Language - Supports 5 languages with automatic detection. 💾 Persistent Settings - Settings saved automatically between sessions. 🛡️ Error Protected - pcall error handling throughout for maximum stability. **Nameplate Addon Compatibility:** Full support for Blizzard Default, ElvUI, KUI Nameplates, and TidyPlates. Partial support for Plater (full support in development). *** ## 📋 Slash Commands The addon can be managed using the following slash commands:
/rnd help # Show all available commands
/rnd enable # Enable the addon
/rnd disable # Disable the addon
/rnd toggle # Toggle the addon on/off
/rnd status # Show current addon status
*** ## 🎮 Compatibility The War Within (Retail) | ✅ Fully Supported (Interface 110105) Mists of Pandaria Classic | ✅ Fully Supported (Interface 50500) Cataclysm Classic | ✅ Fully Supported (Interface 40402) Classic Era | ✅ Fully Supported (Interface 11507) *** ## 📥 Installation 1. **Download** RGX | RND from your preferred platform: * [CurseForge](https://www.curseforge.com/wow/addons/remove-nameplate-debuffs) * [GitHub](https://github.com/donniedice/remove_nameplate_debuffs) * [Wago.io](https://addons.wago.io/addons/rnd) * [WoWInterface](https://www.wowinterface.com/downloads/info26320) 2. **Extract** to your WoW `Interface/AddOns` directory for your version: * **Retail WoW:** World of Warcraft/retail/Interface/AddOns * **Classic/Mists:** World of Warcraft/classic/Interface/AddOns * **Classic Era:** World of Warcraft/classic_era/Interface/AddOns 3. **Restart** WoW and enable the addon in the AddOns menu. ![RealmGX Kiwi](https://raw.githubusercontent.com/donniedice/SimpleQuestPlates/main/images/kiwi.gif) **The Kiwi Says:** "Clean nameplates in seconds!
