# v3.2.0

- Renamed addon folder to `RemoveNameplateDebuffs` (no underscores) and updated the branded title to `Remove Nameplate Debuffs!` with the new orange highlight on the `!`.
- Added a minimap icon with draggable positioning around the minimap ring.
- Added `/rnd icon on` / `/rnd icon off` slash commands to show or hide the minimap icon.
- Added Ctrl+Right-click on the minimap icon as a shortcut to hide it (re-show with `/rnd icon on`).
- Updated the minimap tooltip to surface the left-click toggle, left-drag move, and Ctrl+Right-click hide actions.
- Added `STATUS_MINIMAP` to `/rnd status` so the minimap icon state is visible at a glance.
